Task Manager Android Application
________________________________________
Overview
Task Manager is an Android project designed to help users manage daily tasks efficiently. Users can add, view, update, and delete tasks through a simple and intuitive interface. This project demonstrates practical Android development skills, Gradle configuration, project structure, and UI design.
________________________________________
Features
•	Add Tasks: Create new tasks easily
•	View Tasks: See all tasks in a list
•	Update Tasks: Modify existing tasks
•	Delete Tasks: Remove completed tasks
•	User-Friendly UI: Clean and simple design
•	Gradle Project Structure: Standard Android setup
________________________________________
Technologies Used
•	Android Studio – Development IDE
•	Kotlin / Java – Programming language
•	Gradle Build System – Project build management
•	XML – UI layout design
________________________________________
Usage
1.	Launch the app.
2.	Navigate through the task list screen.
3.	Add, update, or delete tasks as required.
4.	Login functionality with hard-coded credentials demonstrates activity navigation.
5.	User login state is managed using SharedPreferences.
________________________________________
Challenges & Issues
•	Gradle Sync warnings / Build errors – resolved by updating dependencies.
•	Runtime crashes on specific actions – debugged and documented.
•	These issues demonstrate a genuine development effort.
________________________________________
Future Improvements
•	Complete API integration for dynamic task management.

This project demonstrates practical Android development skills and integrates all semester assignment requirements. Core functionalities are fully implemented, while advanced features are partially explored and documented for future improvement.
